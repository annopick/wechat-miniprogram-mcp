- **入口与初始化**：`src/index.ts` 作为应用入口，负责校验环境变量（如 `WECHAT_DEVTOOLS_PORT`），初始化控制 API 配置并启动 MCP 服务器。
- **MCP 服务层**：`src/server.ts` 封装了 `@modelcontextprotocol/sdk`，通过 `StdioServerTransport` 建立通信，并统一分发 `ListTools` 和 `CallTool` 请求到具体的工具处理器。
- **工具模块划分**：工具逻辑分为 `src/tools/control`（基于 HTTP API 的控制操作，如预览、上传）和 `src/tools/auto`（基于自动化 SDK 的交互操作，如页面导航、组件点击）。
- **API 适配层**：`src/api/control.ts` 提供了对微信开发者工具 HTTP 接口的统一调用封装，处理超时、响应类型解析（JSON/文本/二进制图片）及错误重试。
- **构建与发布**：使用 `tsup` 进行 ESM 格式打包，并通过 `scripts/check-shebang.js` 确保生成的 CLI 入口文件包含且仅包含一个正确的 shebang 行，以支持全局命令执行。
